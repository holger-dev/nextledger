# NextLedger

A Nextcloud-native bookkeeping app for freelancers and small teams. Track customers, cases, offers, invoices, products, fiscal years, and GUEB (EUE) inside Nextcloud.

## Abschlagsrechnungen
NextLedger supports advance and final invoices for long-running projects.

- Mark an offer as accepted (Status: "Angenommen").
- Create an advance invoice and link it to the accepted offer.
- Optional service period (start/end) is required for advance invoices.
- The invoice PDF shows the type (Abschlagsrechnung/Schlussrechnung) and an optional offer reference.
