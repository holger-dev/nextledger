# NextLedger Structure

- `appinfo/`: App metadata, routes, bootstrap.
- `lib/Controller/`: Web and API controllers.
- `lib/Db/`: Entities, mappers, and database migrations.
- `lib/Service/`: Domain services (accounting, CRM, invoices, offers, reporting).
- `lib/Migration/`: Nextcloud migrations.
- `templates/`: Server-rendered templates.
- `js/`, `css/`: Frontend assets.
- `tests/`: Unit/integration tests.
